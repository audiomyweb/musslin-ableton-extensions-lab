# CORVEN by MÜSSLIN

Advanced MIDI arpeggio engine for Ableton Live 12 Extensions SDK.

## Description

**CORVEN by MÜSSLIN** is an experimental MIDI arpeggio generator designed for Ableton Live 12.

It creates musical arpeggio ideas directly inside the selected MIDI clip, with a simple interface and a deeper internal engine that handles rhythm, movement, silence, accents, octaves, velocity and note length automatically.

CORVEN is not a static chord tool.  
It is built to generate movement, tension, melodic flow and evolving MIDI phrases that can be used quickly in real production sessions.

The concept is simple: the user chooses a few musical directions, then CORVEN proposes four variations. The producer can insert the chosen idea directly into the active MIDI clip, randomize new ideas, or create a similar variation based on the selected result.

## Main Features

- Automatic MIDI arpeggio generation
- 4 musical variations proposed
- Direct insertion into the selected MIDI clip
- Visual preview matching the real notes copied into the clip
- Simple controls: Root, Style, Emotion, Length
- Randomize button to generate new ideas
- Session memory for the latest settings
- Make Similar function to create a variation close to the selected one
- Automatic note velocity and note length generation
- Musical handling of silences, accents, octaves and micro-variations

## Workflow

1. Select a MIDI clip in Ableton Live.
2. Open CORVEN from the Ableton Live Extensions menu.
3. Choose the Root, Style, Emotion and Length.
4. Generate four arpeggio variations.
5. Preview the musical shape visually.
6. Insert the chosen variation directly into the selected MIDI clip.
7. Use Randomize or Make Similar to explore new ideas.

## Design Philosophy

CORVEN is designed to stay simple on the surface while carrying more musical intelligence inside the engine.

The goal is not to overwhelm the producer with too many controls, but to create fast, expressive and usable MIDI material with a clean workflow.

It is made for electronic music producers who want instant inspiration, controlled variation and musical movement inside Ableton Live.
