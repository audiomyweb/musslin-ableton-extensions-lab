# CORVEN by MÜSSLIN

Advanced Ableton Live 12 Extensions SDK concept by **MÜSSLIN**.

## Concept

**CORVEN by MÜSSLIN** is an experimental MIDI arpeggio engine for Ableton Live 12, designed to generate musical, expressive and directly usable MIDI variations inside a selected clip.

The goal is to keep the interface simple while allowing the internal engine to handle deeper musical behavior: silences, accents, octaves, note length, velocity and subtle micro-variations.

CORVEN is not designed as a static chord pad generator.  
It focuses only on **MIDI arpeggios**, movement, rhythm, musical phrasing and creative variation.

## Main Features

- Automatic MIDI arpeggio generation
- 4 musical variations proposed
- Direct insertion into the selected MIDI clip
- Visual preview matching the real notes copied into the clip
- Simple controls: Root, Style, Emotion, Length
- Randomize button to generate new ideas
- Memory of the latest settings during the session
- “Make Similar” function to create a variation close to the selected one
- Automatic note velocity and note length generation
- Musical handling of silences, accents, octaves and micro-variations

## Workflow

1. Select a MIDI clip in Ableton Live.
2. Choose the basic musical direction with Root, Style, Emotion and Length.
3. Generate 4 arpeggio variations.
4. Preview the variations visually.
5. Choose the one that feels right.
6. Insert it directly into the selected MIDI clip.
7. Use Make Similar to create a related variation without losing the original musical idea.

## Design Philosophy

CORVEN is built around a simple idea:

The user should not fight with too many controls.  
The engine should carry the complexity, while the interface stays clear, fast and musical.

The tool is designed for producers who want quick inspiration, but with coherent MIDI results that can actually be used in a real Ableton Live session.
