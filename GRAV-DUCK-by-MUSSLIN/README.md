# Grav Duck by Müsslin

Version 0.3.0

Anciennement ChordUCK, renommé en Grav Duck by Müsslin.

## Fonctions

- Générateur d’accords MIDI puissant.
- Moteur Expressive Chord ajouté.
- Piano-roll actif : déplacement des notes à la souris avant écriture.
- Mode Arpeggiator : Up, Down, UpDown, Random.
- 4 ou 8 bars.
- Mémoire des réglages et des notes éditées.
- Logo Grav Duck intégré.
- Écriture directe dans le clip MIDI sélectionné.

## Installation

```bash
cd ~/Desktop/grav-duck-by-musslin-v030
npm install
```

## Lancement

```bash
npm run start -- --live "/Applications/Ableton Live 12 Beta.app/Contents/Helpers/ExtensionHost/ExtensionHostNodeModule.node"
```

## Package

```bash
npm run package
```
